## 安装 AI 开发工具

确认 Node.js 版本为 18+ 后，可以安装以下工具：

### Claude Code

```powershell
npm install -g @anthropic-ai/claude-code
```

### Gemini CLI

```powershell
npm install -g @google/generative-ai-cli
```



## 验证安装

```powershell
# 检查 Claude Code
claude-code --version

# 检查工具是否可用
claude-code --help
```
